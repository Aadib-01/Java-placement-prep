# Time & Space Complexity

## What is Time Complexity?
Time Complexity measures how the running time of an algorithm grows as the input size (n) increases.(The rate at which the time increases is called as time complexcity)

## Common Complexities

- O(1) → Constant
- O(log n) → Binary Search
- O(n) → Single loop
- O(n log n) → Merge Sort
- O(n²) → Two nested loops

## Space Complexity

Space used by an algorithm.(Memory space that the program takes)

Total Space =
- Input Space +
- Auxiliary Space

## Interview Tips

- Always mention Time Complexity.
- Then mention Space Complexity.
- Prefer better Time Complexity over saving a little memory.

Example:

Linear Search
Time: O(n)
Space: O(1)

Binary Search
Time: O(log n)
Space: O(1)
